# 题目描述
```
给定一个非负整数数组，你最初位于数组的第一个位置。

数组中的每个元素代表你在该位置可以跳跃的最大长度。

判断你是否能够到达最后一个位置。

示例 1:

输入: [2,3,1,1,4]
输出: true
解释: 我们可以先跳 1 步，从位置 0 到达 位置 1, 然后再从位置 1 跳 3 步到达最后一个位置。
示例 2:

输入: [3,2,1,0,4]
输出: false
解释: 无论怎样，你总会到达索引为 3 的位置。但该位置的最大跳跃长度是 0 ， 所以你永远不可能到达最后一个位置。
```


# 解题思路
- 贪心算法


# 解法1-贪心算法
## 关键点
1. **核心逻辑**：遍历数组，实时维护一个“当前能到达的最大索引”（`maxReach`）；
2. **更新规则**：对于每个位置 `i`，`maxReach = max(maxReach, i + nums[i])`（当前位置能到达的最大范围 = 之前的最大范围 和 当前位置+跳跃长度 的较大值）；
3. **终止条件**：
    - 若 `maxReach >= nums.length - 1`：直接返回`true`（已能到达最后一个下标）；
    - 若 `i > maxReach`：返回`false`（当前位置超出能到达的范围，无法继续前进）。


## 代码
```Java
class Solution {
    public boolean canJump(int[] nums) {
        int maxReach = 0;  // 当前能到达的最大索引
        for (int i = 0; i < nums.length; i++) {
            if (maxReach < i) return false;
            if (maxReach >= nums.length - 1) return true;   // 剪枝
            maxReach = Math.max(maxReach, i + nums[i]);
        }

        return true;
    }
}
```


# 解法2-递归
## 关键点
- 递归尝试从当前位置跳到所有可能的下一步（`1`到`nums[i]`步）；
- 若递归到最后一个下标，返回`true`；
- 若所有路径都尝试过仍未到终点，返回`false`。

## 代码
```Java
class Solution {
    public boolean canJump(int[] nums) {
        // 可能往回跳动，需要记录避免
        boolean[] visited = new boolean[nums.length];
        return dfs(nums, 0, visited);
    }

    private boolean dfs(int[] nums, int i, boolean[] visited) {
        if(i >= nums.length - 1) {
            return true;
        }
        if(i < 0) {
            return false;
        }

        if(visited[i] == true) {
            return false;
        }

        visited[i] = true;

        boolean can = false;
        for(int j = nums[i]; j >= 1; j--) {
            can = can || dfs(nums, i + j, visited);
            can = can || dfs(nums, i - j, visited);
        }

        return can;
    }
}
```


## 复杂度
- 时间复杂度：O(N)
- 空间复杂度：O(1)
