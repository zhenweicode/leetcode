# 题目描述
```$xslt
中位数是有序列表中间的数。如果列表长度是偶数，中位数则是中间两个数的平均值。

例如，

[2,3,4] 的中位数是 3

[2,3] 的中位数是 (2 + 3) / 2 = 2.5

设计一个支持以下两种操作的数据结构：

void addNum(int num) - 从数据流中添加一个整数到数据结构中。
double findMedian() - 返回目前所有元素的中位数。
示例：

addNum(1)
addNum(2)
findMedian() -> 1.5
addNum(3) 
findMedian() -> 2
进阶:

如果数据流中所有整数都在 0 到 100 范围内，你将如何优化你的算法？（treemap记录数值+数量，每次取出来计算中位数）
如果数据流中 99% 的整数都在 0 到 100 范围内，你将如何优化你的算法？（假设那99%的值都是101即可）
```

# 解法1-优先队列
## 关键点
利用优先队列

- 用两个堆维护元素：
    - 大顶堆（maxHeap）：存储左半部分元素（较小的一半），堆顶是左半部分最大值；
    - 小顶堆（minHeap）：存储右半部分元素（较大的一半），堆顶是右半部分最小值；
- 维护规则：
    - 两个堆的大小差不超过1（保证中间元素在堆顶）；
    - 大顶堆的堆顶 ≤ 小顶堆的堆顶（保证左半部分 ≤ 右半部分）；
- `addNum`：
    - 先加入大顶堆，再平衡两个堆（若大顶堆 size > 小顶堆 size + 1，将大顶堆堆顶移到小顶堆）；
    - 检查堆顶大小关系，若大顶堆堆顶 > 小顶堆堆顶，交换两者堆顶；
- `findMedian`：
    - 若总元素数为奇数，返回大顶堆堆顶（左半部分多一个元素）；
    - 若为偶数，返回（大顶堆堆顶 + 小顶堆堆顶）/ 2.0。


## 代码
```Java
public class MedianFinder {
    // 大顶堆：存储左半部分元素（较小的一半），堆顶是左半最大值
    private PriorityQueue<Integer> maxHeap;
    // 小顶堆：存储右半部分元素（较大的一半），堆顶是右半最小值
    private PriorityQueue<Integer> minHeap;

    public MedianFinder() {
        maxHeap = new PriorityQueue<>((x, y) -> y - x);
        minHeap = new PriorityQueue<>();
    }

    public void addNum(int num) {
        // 1. 先加入大顶堆
        maxHeap.offer(num);

        // 2. 平衡两个堆的大小（大顶堆最多比小顶堆多1个元素）
        if (maxHeap.size() > minHeap.size() + 1) {
            minHeap.offer(maxHeap.poll());
        }

        // 3. 保证大顶堆堆顶 ≤ 小顶堆堆顶（左半部分 ≤ 右半部分）
        if (!minHeap.isEmpty() && maxHeap.peek() > minHeap.peek()) {
            int temp = maxHeap.poll();
            maxHeap.offer(minHeap.poll());
            minHeap.offer(temp);
        }
    }

    public double findMedian() {
        int totalSize = maxHeap.size() + minHeap.size();
        if (totalSize % 2 == 1) {
            // 奇数：大顶堆堆顶是中位数
            return maxHeap.peek();
        } else {
            // 偶数：两个堆顶的平均值
            return (maxHeap.peek() + minHeap.peek()) / 2.0;
        }
    }
}
```


## 复杂度
- 时间复杂度：O(logN)，优先队列的出队入队操作都是对数级别的，数据在两个堆中间来回操作是常数级别的
- 空间复杂度：O(N)，使用了三个辅助空间，其中两个堆的空间复杂度是 $O(\cfrac{N}{2})$，一个表示数据流元素个数的计数器 count，占用空间 O(1)
